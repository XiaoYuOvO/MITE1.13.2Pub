# MITE1.13.2
**_一个人不可能只是搞了几个胡萝卜然后制成夜视药水，去到地底世界挖钻石就一夜暴富_**
## 特性
* 让你更难采集资源
* 怪物更具有攻击性
* 续航性更差
* 添加多种怪物、食物、工具、护甲
# 关于
此版本的Minecraft基于MinecraftForge提供的原版1.13.2的源代码及其用于反混淆的srg文件修改而成，
因此它继承了MinecraftForge的LGPL2.1许可证。
